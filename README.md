   # Habit Tracker API

   Базовый REST API для управления привычками: создание, отметка выполнения и получение списка.

   # Возможности

   - Добавление новой привычки
   - Отметка выполнения привычки за текущий день
   - Получение списка всех привычек с датами выполнения
   - Валидация данных (система ожидает строго определенный формат данных)
   - Защита от повторной отметки в один день и обработка прочих ошибок

   # Требования

   - Python 3.8+
   - Виртуальное окружение (рекомендуется)

   # Установка и запуск

   1. Клонируйте репозиторий и перейдите в папку проекта:
      cd API_proj1

   2. Создайте и активируйте виртуальное окружение:

      python -m venv .venv
      source .venv/bin/activate  # macOS/Linux
      # или
      .venv\Scripts\activate     # Windows

   3. Установите зависимости:

      pip install -r requirements.txt

   4. Запустите сервер:

      uvicorn habit_tracker.main:app --reload

   - API будет доступно по адресу http://localhost:8000 
   - Документация (Swagger) — http://localhost:8000/docs

   # Эндпоинты

   1. Создать привычку — POST /habits/
      Тело запроса:
      {
      "name": "Бег"
      }
      Успешный ответ (201 Created):
      {
      "id": 1,
      "name": "Бег"
      }

   2. Отметить выполнение — POST /habits/{id}/mark/
      Пример — POST /habits/1/mark/
      Успешный ответ (200 ОК):
      {
      "id": 1,
      "name": "Бег",
      "last_marked_at": "2025-11-12"
      }

   3. Получить список привычек — GET /habits/
      Успешный ответ (200 ОК):
      [
      {
      "id": 1,
      "name": "Бег",
      "marks": ["2025-11-12", "2025-11-11"]
      }
      ]

   # Хранение данных

   Данные хранятся в оперативной памяти, а после перезапуска сервера все привычки сбрасываются

   # Тестирование
